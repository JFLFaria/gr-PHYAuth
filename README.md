# gr-PHYAuth

GNU Radio 3.8 Module developed for the thesis SDR for physical layer authentication

## Dependencies

- [GNU Radio 3.8](https://github.com/gnuradio/gnuradio/tree/maint-3.8)

## How to install

'''
git clone https://github.com/JFLFaria/gr-PHYAuth.git
cd gr-PHYAuth
mkdir build
cd build
cmake ..
make
sudo make install
sudo ldconfig
'''
